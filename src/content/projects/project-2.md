## Project Overview

Mary UI Laravel Starter Kit is an opinionated, production-ready foundation for building modern web applications. It combines the power of Laravel with the elegance of Mary UI and Livewire Volt to provide developers with a head start on their projects.

## What's Included

### Authentication & User Management
- Complete authentication system with email verification
- User profile management with avatar upload
- Role-based access control (RBAC)
- Two-factor authentication support

### Developer Experience
- **Livewire Volt**: Modern, file-based Livewire components
- **Mary UI Components**: Beautiful, accessible UI components built on DaisyUI
- Pre-configured development tools (Pint, Larastan, Pest)
- Hot module replacement for rapid development

### Production Ready
- Optimized asset bundling with Vite
- Database migrations and seeders
- Email templates with Mailpit integration
- Comprehensive test suite

## Tech Stack Deep Dive

Built on **Laravel 11**, this starter kit leverages the latest framework features including:
- Improved routing and middleware
- Enhanced database query performance
- Native enum support
- Modern PHP 8.3 features

The UI layer uses **Mary UI** components, which are built on top of **DaisyUI** and **Tailwind CSS**, providing a consistent, accessible, and beautiful design system.
